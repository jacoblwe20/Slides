Why use CSS3 Animations

  • Css3 Animations are very smooth and can do alot more, faster then javascript animations (less code).

  • They degrade gracefully because they do not effect, in any way, browsers that do not support

  • Easliy Detectable in Javascript

  • CSS3 Animations are hardware accelerated in mobile safari.

  • Most importantly it allows you to free up your javascript to do more important task.

CSS3 Animation R

Css3 animations are animations that are driven with css.

Hover

There is two ways to do CSS3 animations.

Animation 

<style type="css/text">
@keyframes resize {
	0% {
		padding: 0;
	}
	50% {
		padding: 0 20px;
	}
	100% {
		padding: 0 100px;
	}
}
	
#box {
	animation-name: resize;
	animation-duration: 1s;
	animation-iteration-count: 4;
	animation-direction: alternate;
}	
</style>

Dont Forget Vendor Prefixes

Transitions 

<style type="css/text">
  .animate_all{
    -webkit-transition: all .5s;
    -moz-transition: all .5s;
    -o-transition: all .5s;
    -ms-transition: all .5s;
    transition: all .5s;
  }
  .animate_opacity{
    -webkit-transition: opacity 500ms;
    -moz-transition: opacity 500ms;
    -o-transition: opacity 500ms;
    -ms-transition: opacity 500ms;
    transition: opacity 500ms;
  }
</style>

and will be used for the rest of the demo.

Transition by Classes 

<style type="css/text">
  .demo_two{
    opacity: .5;
    box-shadow: 0 0 1px #06C;
    transition: all 1000ms;
  }
  
  .demo_two.clicked{
    opacity: 1;
    box-shadow: 0 0 10px #06C;
  }
</style>

I add the changes that Id like to animate to. Then I add the class using javascript.

Adding the Class

<script>
//uisng html 5
  var ele = document.querySelector('.demo_two');
  
  ele.addEventListener('click', function(){
      ele.className = ele.className + ' clicked';  
  });
</script>

Differnt Methods of Applying Transition

  • Adding Transition directly to element styles
    Always animated on any style change
  • Adding a differnt animation paramters to added class
    Changing animations parameters on class toggle.
  • Only Adding animation when element need to be animated
    Having a dedicated animation class
