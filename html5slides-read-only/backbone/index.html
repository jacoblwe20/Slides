What is Backbone?

Backbone.js gives structure to web applications by providing models with key-value binding and custom events, collections with a rich API of enumerable functions, views with declarative event handling, and connects it all to your existing API over a RESTful JSON interface.

* backbonejs.org

  • Backbone is a framework that handles data , which allows you to seemlessly sync and display data on a website all in javascript.

  • One Requirement Underscore.js

    • Utility Belt

Why is it cool

  • Dad owns a car dealership

  • Backbone is pure javascript and works well with frameworks like jQuery & Mootools

  • its Open Source

  • the future

What you can do?

  • You can create, store, and display data on the fly in the browser

  • Intergrate with exsisting databases using a RESTFUL api

  • Can be packaged with `PhoneGap` `Cordova` to handle data for mobile apps

How to install

Go to Backbone.js Homepage & download file. Then download Underscore.js. Add them to a subdirectory in your app and include the scripts.

<script src="path-to/underscore.js"></script>
<script src="path-to/backbone.js"></script>

Thats it!
