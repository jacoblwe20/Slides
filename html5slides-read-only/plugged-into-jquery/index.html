Chaining Variables

Chaining is to link variables together using commas as if variable are in an array.

var x = 1, 
  y = 2,
  z = x + y,
  addAll = function(){
  	return x + y + z
  };
		
  document.write(addAll()); 
  // results 3

Following variable can access data from previous variables.

Ternary Logic

This is a simple way to use if else statement while declaring variable

var x = null,
    y = (x === null) ? 2 : 3;
document.write(y) // result 2;
  • inside bracket are the if statement
  • ? is to declare the start of the results
  • : is equal to else

This can be used for multiple layers of logic

Self Executing

//The most common & will be processed in compressor
(function(){/*do stuff*/}());

//Hipster
+function(){/*do stuff*/}();

// Facebooks way
!function(){/*do stuff*/}();

Now these functions will execute by itself due to the parenthese at the end of the function.

Namespace

When making a plugin for multiple users dont take up to common names spaces. Lets say that you have a super cool plugin that puts a random slide picture in a element. Well if you used `$(ele).slide()` it could conflict. With lets say a plugin for a slider.

Multiple Namespace

A plugin that has `$(ele).slide()`, `$(ele).sliding()`, `$(ele).slider()`, and `$(ele).slid()` is too much! There can be so many conflicting plugins. Ok for internal use, but not for mass use

//Self Executing Anonymous Function Wrap
(function($){
   // Data object
   var data = {},
       count = 0,
       //String data for alert, using variable chaining
        h = "Hello, ", str, href;
    //Declaring $().helloWorld
    $.fn.helloWorld = function(opts){
        //Looping through all elements passed to plugin
        var options = {pre : h, multi : '.. again'};
        if(typeof opts === 'object'){
          $.extend(options, opts);                                                
        }
        
        $(this).each(function(i){
            //Binding click to all elements
            $(this).bind('click', function(){
                //Graping the HREF attr
                href = $(this).attr('href');
                //Declaring String to pass to alert
                str = (typeof data[href] === 'boolean') ? options.pre + href + options.multi : options.pre + href;
                //Setting data[element.index] to true to signify the first hello has been called
                data[href] = true;
                // Alerting the created String
                alert(str);
                //Returning False to prevent default anchor behavior
                return false;
            });
        });
        count += 1;
       //Returning this to allow chaining on plugin             
       return this;
    };
//Closing and Calling Anonymous Function
}(jQuery));
